javaScript

JS是什么

JS是一种基于对象 和 事件驱动 的 客户端脚本语言

JS的基本结构

<script>...</script>标签对，将JavaScript脚本代码进行封装，同时告诉浏览器，这是JavaScript代码

language属性：指定封装代码的脚本语言及版本

type属性：指定插入的脚本代码类型

引用Javascript的方式

  1. 使用<script>...</script>标签对

  1. 链接外部的JavaScript文件

DOM和BOM

一、DOM

DOM是什么

文档对象模型，是根据html标签生成的一个对象，可以用来操作页面的元素，顶级对象是document

二、BOM

BOM是什么

BOM（Browser Object Model）即浏览器对象模型，它提供了独立于内容而与浏览器窗口进行交互的对象，其核心对象是 window

BOM的构成

BOM 比 DOM更大，它包含了 DOM，详细可以看看DOM树和BOM的结构图

这里特别说明一下BOM对象下的几个方法：(下面会有具体的使用可以看 - 定时器)

方法 描述
setTimeout(调用函数，[延迟的毫秒数]); 在指定的时间间隔之后执行一次指定的函数
setInterval(回调函数，[间隔的毫秒数]) 在指定的时间间隔重复执行指定的函数

完整写法是window.setInterval 另一个同理，但是window.一般是省略掉的

注意：倒计时效果就是使用setInterval完成的

var和let的区别

  1. var可以先使用在声明（不合理）, let 则不可以会报错
  2. var声明过的变量可以重复声明（不合理），let也不可以

Math

  • 介绍: Math对象是JavaScript提供的一个 “数学”对象
  • 作用: 提供了一系列做数学运算的方法
  • Math对象包含的方法有：
    • random: 生成0 - 1 之间的随机数（包含0 不包含1）
    • ceil: 向上取整
    • floor: 向下取整
    • round: 四舍五入
    • max: 找最大数
    • min: 找最小数
    • pow: 幂运算
    • abs: 绝对值
    • 具体的可以查文档

jQuery

首先要做到它是什么，

  1. jQuery 是一个 JavaScript 函数库
  2. jQuery 是一个轻量级的”写的少，做的多”的 JavaScript 库

jQuery语法

基础语法： $(selector).action()

  • 美元符号定义 jQuery
  • 选择符（selector）”查询”和”查找” HTML 元素
  • jQuery 的 action() 执行对元素的操作

实例：

  • $(“p”).hide() - 隐藏所有

    元素

  • $(this).hide() - 隐藏当前元素

选择器

  1. jQuery 选择器允许您对 HTML 元素组或单个元素进行操作。
  2. jQuery 选择器基于元素的 id、类、类型、属性、属性值等”查找”（或选择）HTML 元素。 它基于已经存在的 CSS 选择器，除此之外，它还有一些自定义的选择器。

AJAX

什么是AJAX

AJAX = 异步 JavaScript 和 XML。

AJAX 是一种用于创建快速动态网页的技术。

通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。
